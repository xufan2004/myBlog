# TS中的类型

# 类型声明

  • 类型声明是TS非常重要的一个特点
  • 通过类类型声明可以指定TS中变量(参数，形参)的类型
  • 指定类型后，当变量赋值时，TS编译器会自动检查值是否符合类型声明，符合则赋值，否则报错
  • 简而言之，类型声明给变量设置了类型，使得变量只能存储魔种类型的值
  • 语法
let 变量:类型;
function fn(参数:类型,参数:类型):类型{

}

# 类型

number // 任意数字
string // 任意字符串
boolean // true OR false
字面量 // 值就是字面量本身
any // 任意类型什么样的值都可以
unknown // 表示未知的值
void // 没有值 || undefined (常用来在方法上使用)
never // 不可以有任何值 (常用来在方法上使用)
object // 任意的js对象
array // 任意的js数组
tuple // 固定长度的数组
enum // 枚举
